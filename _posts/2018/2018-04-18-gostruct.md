Golang の `struct` を Graphviz を用いて可視化する．

[noyuno/gostruct: Golang `struct` graph visualizer](https://github.com/noyuno/gostruct)

例として，次のコードがあったとき，

~~~go
package example

type E struct {
    key int
    val []rune
    D   *D
}

type D struct {
    E *E
}

type C struct {
    E E
}

type B struct {
    D *D
}
type A struct {
    B1 B
    C2 []*C
    D6 *D
    E9 []E
    *C
}
~~~

次のコマンドを実行すると

~~~
gostruct -b1 example A | dot -Tpng -o/tmp/a.png
~~~
